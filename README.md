# SkillSight AR

XR accessibility training project built for Tigerverse 2026.

## 🚀 Overview
SkillSight AR is an augmented reality experience designed to help users understand and interact with accessibility challenges in real-world environments.

## 🧠 Features
- Real-time AR interaction
- Accessibility-focused scenarios
- Interactive learning experience

## 🛠 Tech Stack
- Snap Lens Studio / AR
- TypeScript / JavaScript

## 📦 How to Run
1. Open project in Lens Studio
2. Connect device (Snap Spectacles / phone)
3. Run and test the experience
